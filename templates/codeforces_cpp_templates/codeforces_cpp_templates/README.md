# Codeforces C++17 Algorithm Templates

- Source basis: uploaded common algorithm template PDF.
- All `.cpp` files are standalone C++17 files.
- Graph and DP templates use 0-indexed nodes/states unless a problem-specific conversion is needed.
- Template code intentionally contains no inline comments.
